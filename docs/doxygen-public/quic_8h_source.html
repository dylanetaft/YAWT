65// Quic connection initialization goes like this
66// Initial -> Handshake -> 1-RTT
67// Initial - Establishes header protection key + Client\Server Hello + CIDs
68// Handshake - Establishes RTT keys + Handshake messages
69// 1-RTT - Application keys
