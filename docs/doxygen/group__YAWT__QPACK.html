YAWT_QPACK_Error_t YAWT_QPACK_huff_decode_byte ( const uint8_t * data,
size_t data_len,
uint8_t * bit_offset,
uint8_t * out_byte,
size_t * advance_bytes )

Decode a single Huffman byte from a bitstream.

Parameters
dataThe input bitstream.
data_lenThe input bitstream length.
bit_offsetPointer to the bit position (0-7) within data[0] at which to start.
out_bytePointer to receive the decoded byte.
advance_bytesPointer to receive the number of whole bytes fully consumed.
Returns
YAWT_QPACK_OK on success, or an error code.
Note
Lazy-inits the tree. On success, *bit_offset is set to the residual bit position within the last byte touched. Callers decoding a stream should advance their data pointer by *advance_bytes and carry *bit_offset into the next call.

Encode a single byte into a Huffman bitstream.

Parameters
in_byteThe byte to encode.
outOutput buffer.
out_sizeOutput buffer size (must be >= 4).
bit_offsetPointer to the bit position (0-7) within out[0] at which to start writing.
advance_bytesPointer to receive the number of whole bytes fully completed.
Returns
YAWT_QPACK_OK on success, YAWT_QPACK_ERR_SHORT_BUFFER if out_size < 4.
Note
On success, *bit_offset is set to the residual bit position within the last byte written. Callers building a stream should advance their out pointer by *advance_bytes and carry *bit_offset into the next call.
