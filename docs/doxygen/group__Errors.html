enum YAWT_Err_t

Unified error codes for QUIC and HTTP/3.

Note
RFC-defined wire codes first (QUIC transport errors 0x00-0x10, H3 errors 0x0100-0x0110), then internal/application errors (0x10000+). YAWT_ERR_OK (0x00) serves as both success and the QUIC NO_ERROR wire code.
