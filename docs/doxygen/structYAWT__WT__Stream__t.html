Detailed Description

Per-stream WT state.

Note
Hung off the QUIC stream's YAWT_Q_StreamUserData_t[YAWT_UD_WT] slot. Allocated by WT layer when it first sees a WT signal (0x41/0x54). Buffers the session_id varint which may span multiple QUIC chunks.
