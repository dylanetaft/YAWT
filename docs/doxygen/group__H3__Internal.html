◆ YAWT_qpack_decode_header_block()

YAWT_QPACK_Error_t YAWT_qpack_decode_header_block ( const uint8_t * data,
size_t len,
YAWT_H3_HeaderFields_t * out )

Decode the entire encoded field section (HEADERS frame payload).

Parameters
dataThe encoded data.
lenThe encoded data length.
outThe output header fields section.
Returns
YAWT_QPACK_OK on success, or an error code.
Note
Only static table entries and literal representations are supported. Huffman-encoded strings are decoded into out->huff_scratch (ANB_Blob_t). Name/value pairs are copied into out->slab using the existing header field storage. On error the section may be left partially populated; caller (h3 layer) typically destroys it.

Encode a header fields section into a QPACK header block.

Parameters

bufOutput buffer.
lenOutput buffer length.
writtenPointer to receive the number of bytes written.
Returns
YAWT_QPACK_OK on success, YAWT_QPACK_ERR_SHORT_BUFFER if buf is too small.
Note
Static table only, no Huffman. Writes the 2-byte header block prefix (RIC=0, Base=0) followed by encoded field lines. Uses Indexed representation for full name+value matches, Literal with Name Reference for name-only matches, and Literal with Literal Name for everything else.

Decode a single Huffman byte from a bitstream.

Parameters

out_bytePointer to receive the decoded byte.
