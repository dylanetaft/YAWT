Note
Lifetime: a single global, installed via YAWT_q_con_set_event_handler(). Threading: the QUIC layer is single-threaded (one libev loop). The handler is invoked synchronously from YAWT_q_con_rx() and YAWT_q_con_maintain(). Nothing here is thread-safe.
