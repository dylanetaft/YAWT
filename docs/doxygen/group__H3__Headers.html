◆ YAWT_h3_header_section_size()

size_t YAWT_h3_header_section_size ( const YAWT_H3_HeaderFields_t * headers)

Calculate the field section size per RFC 9114 §10.5.1.

Parameters
headersThe header fields to measure.
Returns
The sum of (name_len + value_len + 32) for each field.
Note
This is the logical size used for SETTINGS_MAX_FIELD_SECTION_SIZE enforcement, not the QPACK-encoded wire size. Use YAWT_qpack_header_block_size() for wire size.
