Detailed Description

Per-event parameters.

Warning
Transience: ALL pointers in this union are valid ONLY for the duration of the handler call. They reference internal/transient storage (the input datagram buffer, a slab item, or an encode scratch buffer) that the QUIC layer reuses immediately after the handler returns. Copy anything you need to retain.
