◆ YAWT_Q_Frame_BufferedStream_t

typedef struct YAWT_Q_Frame_BufferedStream YAWT_Q_Frame_BufferedStream_t

A STREAM frame plus the storage its .frame.data points at.

Note
Used when a frame must outlive the input datagram (out-of-order RX buffering and TX queuing). bf->data is the OWNED inline copy. bf->frame.data is the borrowed pointer, set to point at bf->data when buffered.
