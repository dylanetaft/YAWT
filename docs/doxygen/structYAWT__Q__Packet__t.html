Detailed Description

Flat packet struct — all packet types collapsed into one.

Warning
Transience: when produced by YAWT_q_parse_packet, the pointer fields below (payload, raw, and extra.*.token) borrow INTO the input datagram buffer. They are valid only while that buffer is. Decrypt happens in-place on those bytes. Do not retain these pointers beyond the current parse scope.
