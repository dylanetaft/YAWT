Detailed Description

Stream metadata — one per open stream, stored in the con->stream_userdata slab.

Note
Tracks reassembly + flow-control position so EVT_STREAM can be delivered gap-free. RFC 9000 §3.1/3.2: TX and RX are independent state machines. If the metadata exists, both directions are active by default. The state field tracks termination reasons (FIN, RESET, STOP_SENDING).
