Detailed Description

H3 connection object.

Note
Hung off the QUIC connection's YAWT_UD_H3 slot. Allocated on EVT_CONNECTED, freed on EVT_CLOSE (which con_free guarantees fires once). Per-stream H3 metadata is malloc'd and stored in the QUIC stream's YAWT_Q_StreamUserData_t[YAWT_UD_H3] slot (1:1 with QUIC streams).
