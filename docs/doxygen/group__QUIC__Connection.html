Detailed Description

Typedef Documentation

◆ YAWT_Q_Connection_t

typedef struct YAWT_Q_Connection_t YAWT_Q_Connection_t

A QUIC connection.

Note
Owned by the QUIC layer (created in con_create, destroyed in con_free). Indexed in global CID hash tables; found via con_find_by_cid. Threading: NOT thread-safe — one libev loop drives all connections. All fields are mutated only on that thread, synchronously with con_rx / con_maintain. Most fields are QUIC-internal; user_data is the app's.
