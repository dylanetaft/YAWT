Parameters

Enqueue STREAM frames for the given scatter/gather buffers.

Parameters
conThe QUIC connection.
stream_idThe target stream ID.
iovScatter/gather buffer array.
iov_countNumber of elements in iov.
finIf true, marks the stream finished after the last chunk.
Returns
YAWT_Q_OK on success, or an error code.
Note
Ownership: copies all iov[] data (caller keeps its buffers).
