Detailed Description

Read cursor for zero-copy datagram parsing.

Note
cursor advances as bytes are consumed; data/len are the input buffer. Errors are STICKY. Once err is non-OK, every later decode/parse on this cursor is a no-op. Transience: pointers handed back point INTO data.
