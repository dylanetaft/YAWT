Detailed Description

Generic RFC 9297 capsule protocol parser and encoder.

Note
Capsules are Type-Length-Value structures used by WebTransport and HTTP datagrams. This parser is reusable and not WT-specific.
Buffering strategy:
  • Header phase: accumulate up to 16 bytes for Type + Length varints
  • Payload phase: buffer entire payload for control capsules (max 2KB)
  • Exception: DATAGRAM capsules (type 0x00) stream payload directly without buffering, since datagram payloads can be large.
  • Unknown capsule types are logged and skipped (RFC 9297 §3.2).
Synchronous API: parse_feed() returns status codes. When a capsule is complete, call get_current() to retrieve the data. The parser auto-resets when reaching the end of a capsule length.
