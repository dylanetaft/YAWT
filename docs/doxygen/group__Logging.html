Detailed Description

Compile-time gated logging utilities.

Enable logging by defining YAWT_ENABLE_LOGGING at compile time. Optionally set YAWT_LOG_LEVEL to filter messages (defaults to YAWT_LOG_DEBUG when enabled). When logging is disabled, YAWT_LOG() expands to a no-op.
